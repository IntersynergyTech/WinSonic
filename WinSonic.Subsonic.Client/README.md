## Subconic Client Readme
The client is an autogenerated API client generated from the OpenAPI specification of the Subsonic API. It is generated from the published API documentation available at `https://github.com/opensubsonic/open-subsonic-api` using [OpenAPI Generator](https://openapi-generator.tech/). 

### Regenerating the Client
To regenerate the client, you will need to have [OpenAPI Generator](https://openapi-generator.tech/) installed. You can install it from their website or via a bunch of options (npm, binaries, whatever). Once you have it installed, you can run the following command in the root of this project:

I would also recommend cloning a copy of the OpenAPI spec repo to your machine, as the spec is spread out over a bunch of files, rather than being all in one big spec file.

Once you have the OpenAPI spec repo cloned, you can run the following command in this folder to generate the client:

```bash
openapi-generator-cli generate -i path/to/open-subsonic-api/openapi/openapi.json -g csharp -o . -packageName WinSonic.Subsonic.Client
```
